# 🥗 Comprehensive Analysis and Dietary Strategies with Tableau: A College Food Choices Case Study

## 📖 Project Overview

This project analyzes the dietary habits, lifestyle choices, and
academic performance of college students using Tableau. The objective is
to identify meaningful relationships between food consumption patterns,
nutrition awareness, stress-related eating, and student well-being
through interactive dashboards and visual analytics.

Using Tableau, raw survey data is transformed into dynamic
visualizations that help students, universities, nutritionists, and
cafeteria management make informed decisions to promote healthier
lifestyles.

------------------------------------------------------------------------

## 🎯 Objectives

-   Analyze students' dietary habits.
-   Understand the impact of nutrition on academic performance.
-   Identify stress-related eating patterns.
-   Examine breakfast habits and calorie awareness.
-   Provide data-driven recommendations for healthier eating.

## 📂 Project Workflow

### 1. Data Collection

-   Collected the student dietary survey dataset.
-   Connected the dataset with Tableau.

### 2. Data Preparation

-   Cleaned and transformed data.
-   Removed duplicates and handled missing values.
-   Prepared fields for visualization.

### 3. Data Visualization

Created interactive visualizations including: - Breakfast Habits - Fast
Food Consumption - Fruit & Vegetable Intake - Comfort Food Preferences -
Calorie Awareness - Stress-related Eating - Academic Performance -
Cultural Cuisine Preferences

### 4. Dashboard

-   Interactive filters
-   KPI cards
-   Responsive layout
-   Comparative analysis

### 5. Story

The Tableau Story presents: 1. Student Dietary Overview 2. Breakfast
Habits 3. Fast Food Analysis 4. Stress vs Comfort Food 5. Calorie
Awareness 6. Healthy Food Consumption 7. Academic Insights 8.
Recommendations

### 6. Performance Testing

-   Data loading performance
-   Dashboard responsiveness
-   Filter utilization
-   Calculated fields
-   Visualization performance

### 7. Web Integration

Embedded the Tableau Dashboard and Story into a Flask web application.

## 💡 Real-World Use Cases
